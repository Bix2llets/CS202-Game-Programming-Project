Detailed Description

Manages collections of waypoints for different paths and provides drawing and access methods.

The Map class stores multiple paths, each as a vector of Waypoints, and allows access to them by path number. It also provides methods to construct a map from a vector of waypoints or from a file, and to draw the map.
